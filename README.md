# Surbhi AI OS

Surbhi AI OS is a personal academic productivity system built using Streamlit.

## Features

- AI Study Scheduler
- DSA Task Tracker
- Web Development Tracker
- Academic Subject Planner
- Internship Test Mode
- Exam-aware study planning

## Tech Stack

- Python
- Streamlit
- JSON (data storage)

## How to Run

```bash
pip install streamlit
python3 -m streamlit run app.py
